# Retos — Patrones

Ideas de ejercicios para practicar los conceptos de esta sección.

---

1. **Galería de rayas**: crear tres rectángulos iguales con rayas en tres ángulos distintos (0°, 45°, 90°) usando el mismo patrón base y `patternTransform="rotate(n)"`. Etiquetar cada uno con su ángulo.

2. **Fondo de tarjeta con patrón de puntos**: diseñar una tarjeta (rect con rx) que use un patrón de polka dots como fondo, con el texto superpuesto. El patrón debe verse semitransparente usando `fill-opacity` en los círculos.

3. **Grid técnico**: crear un SVG de diagrama con un grid de fondo usando un patrón de cuadrícula fina (1px, color gris claro) y encima dibujar formas con coordenadas que coincidan con el grid.

4. **Marco con patrón y agujero**: usando un path con subpaths y `fill-rule="evenodd"`, crear un marco (rectángulo exterior con agujero interior) rellenado con un patrón de rayas diagonales. El interior debe quedar vacío.

5. **Marca de agua en diagonal**: crear un patrón que repita el texto "CONFIDENCIAL" en diagonal (patternTransform="rotate(-30)") sobre un fondo blanco. Aplicarlo sobre un documento simulado (varios rectos y líneas de texto).

6. **Patrón con gradiente interno**: definir un gradiente lineal y usarlo como fill de un rectángulo dentro de la celda de un patrón. Aplicar el patrón a un círculo grande. El resultado debe ser rayas con transición de color.
